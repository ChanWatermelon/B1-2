## 🏷 브랜드 개요

- **브랜드명:** ZEROLAG (제로랙) — "지연(lag)을 0으로"
- **카테고리:** 게이밍 특화 에너지 드링크 (제로 슈거)
- **타겟:** 16~29세 코어 게이머 · 스트리머 · e스포츠 지망생
- **톤앤매너:** 하이에너지 · 사이버 네온 · 하이컨트라스트 · 시네마틱
- **컬러:** 일렉트릭 시안 `#00E5FF` · 네온 마젠타 `#FF2E97` · 딥블랙 `#0A0A0F`
- **USP:** 단순 각성이 아니라 "반응속도(0.1초)에 집중을 건다"
- **핵심 메시지:** "결정적인 그 0.1초, 흔들리지 마라 — 지연은 없다, ZEROLAG."

## 🎬 영상 구성 (총 8초 · 단일 명령 1회 생성)

기승전결 구조를 한 프롬프트 안에 타임코드로 배치합니다.

| 구간 | 비트 | 내용 |
|---|---|---|
| 0.0–2.0s | THE LAG | 결정적 순간의 0.1초 지연 (문제) |
| 2.0–3.5s | PICKUP | ZEROLAG 캔 등장 (전환) |
| 3.5–5.0s | OVERCLOCK | 각성 · 집중 스위치 ON |
| 5.0–6.5s | VICTORY | 압도적 승리 (해결) |
| 6.5–8.0s | BRAND LOCKUP | 로고 · 슬로건 · CTA (브랜드 인지) |

전체 통합 프롬프트와 씬별 상세 프롬프트는 **스토리보드 문서 §6**에 있습니다.
