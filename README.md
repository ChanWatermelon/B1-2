# ZEROLAG — AI 기반 브랜드 광고 패키지

게이밍 특화 에너지 드링크 **ZEROLAG**의 AI 생성 브랜드 광고 프로젝트입니다.
텍스트·이미지·비디오·오디오 생성형 AI를 활용해 **10초 이내 광고 영상**과 **스토리보드(기획) 문서**를 제작하는 것을 목표로 합니다.

## 🏷 브랜드 개요

- **브랜드명:** ZEROLAG (제로랙) — "지연(lag)을 0으로"
- **카테고리:** 게이밍 특화 에너지 드링크 (제로 슈거)
- **타겟:** 16~29세 코어 게이머 · 스트리머 · e스포츠 지망생
- **톤앤매너:** 하이에너지 · 사이버 네온 · 하이컨트라스트 · 시네마틱
- **컬러:** 일렉트릭 시안 `#00E5FF` · 네온 마젠타 `#FF2E97` · 딥블랙 `#0A0A0F`
- **USP:** 단순 각성이 아니라 "반응속도(0.1초)에 집중을 건다"
- **핵심 메시지:** "결정적인 그 0.1초, 흔들리지 마라 — 지연은 없다, ZEROLAG."

## 🎬 영상 구성 (총 8초 · 단일 명령 1회 생성)

기승전결 구조를 한 프롬프트 안에 타임코드로 배치합니다.

| 구간 | 비트 | 내용 |
|---|---|---|
| 0.0–2.0s | THE LAG | 결정적 순간의 0.1초 지연 (문제) |
| 2.0–3.5s | PICKUP | ZEROLAG 캔 등장 (전환) |
| 3.5–5.0s | OVERCLOCK | 각성 · 집중 스위치 ON |
| 5.0–6.5s | VICTORY | 압도적 승리 (해결) |
| 6.5–8.0s | BRAND LOCKUP | 로고 · 슬로건 · CTA (브랜드 인지) |

전체 통합 프롬프트와 씬별 상세 프롬프트는 **스토리보드 문서 §6**에 있습니다.

## 🚀 제작 워크플로

1. **키비주얼 확정** — 스토리보드의 이미지 프롬프트로 Midjourney에서 씬별 이미지 생성 (`--sref 2847193045` 고정, 톤 일관성 유지)
2. **영상 생성** — §6 **통합 영상 생성 프롬프트**를 Veo 3 등에 입력해 8초(최대 10초) 1클립 생성
   - Veo 3처럼 오디오 동시 생성 도구는 프롬프트의 `AUDIO` 라인으로 사운드까지 한 번에 생성
3. **오디오 생성** (영상 도구가 사운드 미지원 시) — §9 프롬프트로 Suno(BGM) · ElevenLabs(SFX·내레이션) 별도 생성
4. **립싱크** (보너스) — 인물 이미지 + TTS 음성을 Runway Act-One에 입력
5. **통합 편집** — CapCut에서 자막·오디오 레벨·색보정 후 **16:9 / 9:16** 각각 익스포트
6. **최종 인코딩** — 1080p · 24–30fps · **H.264 / AAC** · MP4
